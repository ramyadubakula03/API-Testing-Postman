# API Testing Project (Postman)

## Description
Performed API testing using Postman including GET and POST requests.

## Features Tested
- GET request (fetch user data)
- POST request (create new data)
- Response validation
- Status code verification

## Negative Testing
- Sent empty body
- Sent invalid data
- Tested invalid endpoint

## Tools Used
- Postman
- JSONPlaceholder API

## Results
- Verified status codes (200, 201, 404)
- Validated API responses
